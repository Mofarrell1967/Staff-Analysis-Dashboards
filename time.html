Dashboard Overview

The time analysis dashboard allows the user to view and filter total time spent based on the following fields.

Staff Name.

Work Type.

Work Priority.

Equipment Platform.

The top panel displays the total number of hours as row-charts and the 2nd panel displays a line graph to show a time-line for the total time spent or specific filtered selections.

Bar Charts

Filtering

To filter the graphs, click on single or multiple bars to show the time data for each selection. Selection choices will be displayed above each graph.

Each filter choice will display the data relevant to the filters and the number of hours for each selection is displayed as a label on top of each graph bar.

Selected or total filter reset

To reset a single filter, click on the selected bar and the filter will be removed.

To remove all filters, click on the 'Reset All' in the top right hand corner of the page.

Line Graph

Filtering

The line graph will display time over months based on the filtering selections in the bar charts.

Users can also filter using the 'brush on' function by mouse click on first time point in the graph and then drag mouse to end point and release. This will filter the data for that selected period

Selected or total line graph filter reset

To remove all filters, click on the 'Reset All' in the top right hand corner of the page. To removed a 'brush on' filter, a mouse click in an unfiltered area of the graph will reset the selection.
