Dashboard Overview

The task analysis dashboard allows the user to view and filter work records based on the following fields.

Staff Name.

Work Type.

Work Priority.

Equipment Platform.

The top panel displays the total number of tasks as row-charts and the 2nd panel displays the record details for the selections.

Bar Charts

Filtering

To filter the graphs, click on single or multiple bars to show the task data for each selection. Selection choices will be displayed above each graph.

Each filter choice with display the data relevant to the filters and the number of selected records will be displayed in the top right hand corner of the page.

Selected or total filter reset

To reset a single filter, click on the selected bar and the filter will be removed.

To remove all filters, click on the 'Reset All' in the top right hand corner of the page.

X-Axis Display

The X-Axis is dynamic and will automatically render ticks to provide the best display of the filtered data.

This ensures that all displays will be easily read for analysis.

Data Table

Data Filtering

The data table will display all records in the database upon loading and the right hand scroll bar can be used to view the records.

Each filter selection on the graphs will automatically filter the data table to those selections and only relevant records will be displayed.

Selected or total filter reset

The data table will undo filters revert to total view as per the reset commands for the graphs.
